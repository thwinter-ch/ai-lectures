# GAMMA Prompt — Wie ich diese Vorlesung erstellt habe

**Dauer:** 15 Minuten | **Folien:** 8–10

---

## Gesamtstil

- Sauber, modern, dunkler oder tiefblauer Hintergrund
- Minimaler Text — der Redner trägt die Geschichte, Folien verankern Kernpunkte
- Icons: Einfach, Monoline-Stil (kein 3D, keine Farbverläufe)
- Keine Stockfotos von Menschen, die am Computer arbeiten (überstrapaziert)
- Akzentfarbe für Hervorhebungen: Elektrisches Blau oder Teal gegen Dunkel
- Schrift: Sans-Serif, modern (Inter, DM Sans oder ähnlich)

---

## Folie 1: Titel

**Titel:** Wie ich diese Vorlesung erstellt habe

**Untertitel:** Mit AI eine Vorlesung über AI erstellen

**Visual:** Einfaches rekursives/Meta-Icon — z.B. eine Folie in einer Folie oder ein Schleifen-Symbol

**Gesprächspunkte:**
- "Diese Vorlesung wurde mit AI erstellt. Lassen Sie mich Ihnen die Belege zeigen."
- Die Meta-Demonstration aufsetzen

---

## Folie 2: Das Dogfooding-Prinzip

**Titel:** Mein eigenes Hundefutter essen

**Layout:** Einzelnes Konzept, grosse Schrift

**Haupttext:** "Wenn Sie AI-Workflows unterrichten, sollten diese Workflows das erstellt haben, was Sie unterrichten."

**Untertext (kleiner):** In der Tech-Branche nennen wir das *Dogfooding* — das eigene Produkt selbst nutzen.

**Visual:** Einfaches Icon — Gabel und Teller oder rekursive Schleife

**Gesprächspunkte:**
- Beweist, dass die Tools funktionieren
- Deckt echte Reibung auf
- Erzwingt Ehrlichkeit darüber, was wirklich hilft

---

## Folie 3: Der Stack

**Titel:** Die Tools, die das hier gebaut haben

**Layout:** 4 horizontale Tool-Karten in einer Reihe

| Tool | Rolle |
|------|-------|
| **Wispr Flow** | Spracherfassung — Ideen beim Spazieren |
| **Perplexity** | Recherche mit Quellenangaben |
| **Claude Code** | Planung + parallele Agenten |
| **GAMMA** | Folien aus Text-Prompts |

**Visual:** Jedes Tool als Icon oder Logo-Platzhalter mit seiner Einzeiler-Funktion

**Gesprächspunkte:**
- Gehen Sie jedes kurz durch
- "Die meisten davon werden Sie heute nutzen"

---

## Folie 4: Von Sprache zu Recherche

**Titel:** Es begann mit Sprache

**Layout:** Einfaches Flussdiagramm (links nach rechts)

```
Spazieren + Denken  →  Wispr Flow  →  Rohe Notizen  →  Perplexity  →  Strukturierte Recherche
```

**Visual:** Icons für jeden Schritt (Person beim Gehen, Mikrofon, Notizblock, Such-Icon, Dokument)

**Gesprächspunkte:**
- Ideen warten nicht auf Tastaturen
- Recherche, die Tage dauerte, braucht jetzt Minuten

---

## Folie 5: Parallele Agenten

**Titel:** Orchestration, nicht Konversation

**Layout:** Zentraler Knotenpunkt mit Speichen

```
          [Claude Code]
              ↓
   ┌─────────┼─────────┐
   ↓         ↓         ↓
Skript    Folien   Übungen
Agent     Agent     Agent
```

**Visual:** Hub-and-Spoke-Diagramm, das parallele Arbeitsströme zeigt

**Schlüsselzitat zur Anzeige:** "Sie sprechen nicht mit einer AI. Sie leiten ein Team."

**Gesprächspunkte:**
- Parallele Ausführung, nicht sequenziell
- Skript-Schreiben während Übungen gebaut werden
- Koordination geschieht automatisch

---

## Folie 6: Der Zeitplan

**Titel:** Zwei Wochen, Hintergrundprozess

**Layout:** Horizontale Zeitleiste

| Datum | Phase | Tools |
|-------|-------|-------|
| 24. Jan, Abend | Sprach-Brainstorming | Wispr Flow |
| 24. Jan, danach | Recherche + Briefing | Perplexity |
| 25.–30. Jan | Parallele Entwicklung (9 Segmente) | Claude Code |
| 31. Jan | Finale Integration | Claude Code + GAMMA |
| 1.–5. Feb | Übersetzung, Korrekturen, Feinschliff | Claude Code |

**Visual:** Zeitleiste mit Punkten und Verbindungslinie, Tool-Icons über jeder Phase

**Gesprächspunkte:**
- Keine Vollzeitarbeit — Hintergrundprozess
- Während der eigentlichen Arbeit wurde das hier gebaut

---

## Folie 7: Wie einen neuen Mitarbeiter einarbeiten

**Titel:** AI leiten — ehrlich

**Layout:** Zwei Spalten

| Die Realität | Was es braucht |
|--------------|----------------|
| Meinungsverschiedenheiten über Struktur | Ihre Standards klar durchsetzen |
| AI produziert Konzernsprache | Ihren Ton kalibrieren |
| Generischer Output ohne Kontext | Domänenwissen einbringen |
| Manchmal: Schimpfwörter | Geduld. Und nochmal Geduld. |

**Schlüsselzitat zur Anzeige:** "AI zu leiten ist wie einen neuen Mitarbeiter einzuarbeiten. Am Anfang ist es Kalibrierung."

**Visual:** Einfache Zwei-Spalten-Darstellung, kontrastierende Farben

**Gesprächspunkte:**
- Der Prozess war nicht harmonisch — das ist normal
- Mit der Zeit wird die Zusammenarbeit besser
- Anekdote: Streitereien mit Claude, Schimpfwörter inklusive

---

## Folie 8: Was sich ändert

**Titel:** Vom Ersteller zum Regisseur

**Layout:** Zwei-Spalten-Vergleich

| Altes Modell | Neues Modell |
|--------------|--------------|
| Manuelle Gliederung | Repository von Bausteinen |
| Langsame Recherche | AI-beschleunigt |
| Einheits-Grösse für alle | Zugeschnitten auf Ihr Publikum |
| Jedes Mal bei Null | Schnelle Iteration auf Bestehendem |

**Schlüsselzitat zur Anzeige:** "Alles ändert sich alle 3 Monate. Jetzt können Sie mithalten."

**Visual:** Einfache Links/Rechts-Aufteilung, kontrastierende Farben (grau vs. leuchtend)

**Gesprächspunkte:**
- Sie bauen Bausteine auf, die wiederverwendbar sind
- Guter Inhalt schlecht verpackt bleibt wirkungslos — AI hilft bei der Storyline
- Anekdote: Freundin in Weiterbildung — spannender Inhalt, einschläfernde Lehrperson

---

## Folie 9: Die ehrlichen Vorbehalte

**Titel:** Nicht besser. Schneller.

**Layout:** Einzelnes Zitat + Liste

**Schlüsselzitat zur Anzeige:** "Mit Ovomaltine kannst du's nicht besser, aber länger. Mit AI ist es ähnlich — schneller, aber nur wenn Sie führen."

**Liste darunter:**
- Ohne Fachkompetenz → generischer Einheitsbrei
- Ohne Urteilsvermögen → AI schliesst alles ein
- Ohne finalen Durchgang → peinliche Fehler

**Visual:** Einfaches Warnungs-Icon, aber mit Augenzwinkern (Ovomaltine-Referenz)

**Gesprächspunkte:**
- Ehrlich bleiben: AI ist ein Werkzeug, kein Ersatz
- Menschliches Urteilsvermögen bleibt der Engpass, der zählt

---

## Folie 10: Was Sie heute bauen werden

**Titel:** Sie sind dran

**Layout:** Drei nummerierte Punkte

1. **Schreibstil-Profil** — AI, die wie Sie klingt
2. **Firmenrecherche-Workflow** — Stunden zu Minuten
3. **Second Brain Setup** — Wissen, das sich kumuliert

**Visual:** Checkliste oder nummerierte Blöcke

**Untertext:** "Sie lernen keine Tools. Sie bauen Infrastruktur."

**Gesprächspunkte:**
- Das sind Grundlagen
- Der Gewinn liegt nicht heute — er liegt an jedem Tag danach
- Überleitung: "Als Nächstes: warum 2026 das Jahr der AI-Agenten ist"

---

## GAMMA Einstellungen

- **Präsentationslänge:** Kompakt (8–10 Folien)
- **Ton:** Professionell, direkt, leicht respektlos
- **Farbschema:** Dunkelmodus, hoher Kontrast (Navy/Anthrazit + elektrischer Akzent)
- **Übergänge:** Einfaches Fade, keine ablenkenden Animationen
- **Schrift:** Sans-Serif, modern

---

## Hinweise für GAMMA Import

Beim Einfügen in GAMMA:

1. Beginnen Sie mit: "Erstelle eine Präsentation darüber, wie diese Vorlesung mit AI-Tools erstellt wurde"
2. Fügen Sie die obige Folien-für-Folien-Struktur ein
3. Spezifizieren Sie: "Minimaler Text, dunkles professionelles Theme, einfache Icons verwenden"
4. Fordern Sie an: "Keine Stockfotos, Diagramme und saubere Icons verwenden"
5. Ergänzen Sie: "Der Ton sollte selbstbewusst und praktisch sein, nicht übertrieben"

GAMMA wird Visuals generieren — prüfen und ersetzen Sie alles, was generisch oder übermässig konzernhaft wirkt.
