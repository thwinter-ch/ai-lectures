# Professionelles Headshot — Prompt-Vorlage

## Basis-Prompt (Gemini)

```
Erstelle ein professionelles Porträtfoto für ein LinkedIn-Profil.

Person: [BESCHREIBUNG: Geschlecht, ungefähres Alter, Haarfarbe, Hautton]
Stil: Professionell, freundlich, nahbar
Kleidung: [Business casual / Formell / Branchenspezifisch]
Hintergrund: [Neutral grau / Büroumgebung unscharf / Einfarbig]
Beleuchtung: Weiches, schmeichelhaftes Studiolicht
Blickrichtung: Direkt in die Kamera, leichtes Lächeln
Bildausschnitt: Kopf und Schultern, etwas Platz über dem Kopf

Keine Logos, kein Text, keine Wasserzeichen.
```

## Beispiel ausgefüllt

```
Erstelle ein professionelles Porträtfoto für ein LinkedIn-Profil.

Person: Frau, Mitte 30, dunkelbraune Haare schulterlang, europäischer Hautton
Stil: Professionell, freundlich, nahbar
Kleidung: Dunkelblaue Bluse, Business casual
Hintergrund: Neutral hellgrau, leicht unscharf
Beleuchtung: Weiches, schmeichelhaftes Studiolicht von vorne-links
Blickrichtung: Direkt in die Kamera, selbstbewusstes aber warmes Lächeln
Bildausschnitt: Kopf und Schultern, etwas Platz über dem Kopf

Keine Logos, kein Text, keine Wasserzeichen.
```

## Varianten

### Kreative Branche
```
Hintergrund: Künstlerisches Atelier unscharf im Hintergrund
Kleidung: Smart casual, etwas Farbe erlaubt
Stil: Kreativ aber professionell
```

### Gesundheitswesen
```
Kleidung: Weisser Kittel oder professionelle medizinische Kleidung
Hintergrund: Helle, saubere Umgebung
Stil: Vertrauenswürdig, kompetent, warmherzig
```

### Konzern / Finanz
```
Kleidung: Dunkler Blazer, formell
Hintergrund: Neutral, fast abstrakt
Stil: Seriös, kompetent, selbstbewusst
```
