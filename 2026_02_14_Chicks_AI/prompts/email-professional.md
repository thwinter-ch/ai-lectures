# Professionelle E-Mails — Prompt-Vorlage

## Basis-Prompt

```
Schreibe eine professionelle E-Mail.

Empfänger: [Name, Rolle, Beziehung zu mir]
Anlass: [Was ist der Grund für die E-Mail?]
Ziel: [Was soll der Empfänger tun/wissen/fühlen?]
Ton: [Formell / Freundlich-professionell / Kollegial]
Länge: [Kurz (3-4 Sätze) / Mittel / Ausführlich]

Kontext: [Relevante Hintergrundinformationen]
```

## Beispiel: Erste E-Mail an neuen Chef

```
Schreibe eine professionelle E-Mail.

Empfänger: Frau Müller, Abteilungsleiterin, meine neue Chefin
Anlass: Erster Arbeitstag morgen, möchte mich kurz vorstellen
Ziel: Positiven ersten Eindruck machen, Vorfreude zeigen
Ton: Freundlich-professionell, nicht zu locker
Länge: Kurz (4-5 Sätze)

Kontext: Ich starte morgen als neue HR-Assistentin. Wir hatten ein gutes Vorstellungsgespräch vor 3 Wochen.
```

## Beispiel: Um Hilfe bitten

```
Schreibe eine professionelle E-Mail.

Empfänger: Thomas, IT-Support, kennen uns noch nicht
Anlass: Brauche Zugang zu einem System
Ziel: Schnelle Hilfe bekommen ohne zu nerven
Ton: Freundlich, respektvoll gegenüber seiner Zeit
Länge: Kurz

Kontext: Ich bin neu (seit 1 Woche), brauche Zugang zum CRM-System für meine Arbeit. Meine Chefin sagte, ich soll mich an IT wenden.
```

## Beispiel: Feedback geben

```
Schreibe eine professionelle E-Mail.

Empfänger: Kollege im Projektteam
Anlass: Seine Präsentation gestern war gut, ein Punkt war aber unklar
Ziel: Positives verstärken + konstruktiv einen Verbesserungsvorschlag machen
Ton: Kollegial, wertschätzend
Länge: Mittel

Kontext: Wir arbeiten seit 2 Monaten zusammen. Er hat gestern die Quartalszahlen präsentiert. Die Visualisierungen waren super, aber die Schlussfolgerungen kamen zu abrupt.
```

---

**Tipp:** Wenn du eine E-Mail-Vorlage findest, die gut funktioniert, speichere sie mit Notizen in deinem Second Brain.
