# Prompt-Bibliothek

Fertige Prompts zum Kopieren und Anpassen. Das ist der Grundstock für dein eigenes Second Brain.

| Datei | Zweck |
|-------|-------|
| `headshot-prompt.md` | Professionelles Profilbild generieren |
| `email-professional.md` | Professionelle E-Mails schreiben |
| `voice-discovery.md` | 10-Fragen-Übung für deinen Schreibstil |
| `document-summary.md` | Lange Dokumente zusammenfassen |
| `onboarding-questions.md` | Fragen für die ersten Wochen im neuen Job |

---

**Tipp:** Wenn ein Prompt gut funktioniert, speichere ihn in deinem Second Brain mit Notizen, was du angepasst hast und warum es funktioniert hat.
