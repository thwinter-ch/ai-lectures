# Prompting Basics

Prompt engineering fundamentals for non-technical users.

## Core Concepts

1. **Context Setting** — Tell the AI who it is and who you are
2. **Task Specification** — Be explicit about what you want
3. **Format Control** — Specify output structure
4. **Iteration** — Refine through conversation

## Common Patterns

- Executive summary generation
- Document comparison
- Meeting prep synthesis
- Policy review and gap analysis
