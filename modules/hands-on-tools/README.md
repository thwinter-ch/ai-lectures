# Hands-On Tools

Practical walkthroughs for AI productivity tools targeting executives and board members.

- **Claude** — Long-form analysis, document summarization, strategic brainstorming
- **ChatGPT** — General purpose, image generation, browsing
- **Perplexity** — Research, citations, fact-checking
- **Copilot** — Office integration, email drafting

## Use Cases

- Board briefing preparation
- Meeting memo generation
