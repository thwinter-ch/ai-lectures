## Usage

Each module contains:
- `README.md` — Overview and learning objectives
- `slides/` — Presentation materials
- `exercises/` — Hands-on activities
- `docs.gitignore/` — Private materials (speaker notes, full versions)

Modules can be assembled into lectures by copying or referencing.
